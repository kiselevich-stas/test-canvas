# test-canvas — версия со стором

В этой версии приложения вся общая логика перенесена в единый стор `src/store/graphStore.js`.

## Что изменено

- поиск в сайдбаре теперь изменяет граф через стор
- статистика наверху считает данные из стора
- выбранный узел и активная вкладка правой панели хранятся в сторе
- компоненты связаны между собой не через локальные состояния, а через единый API стора
- добавлены JSDoc-комментарии на русском для ключевого API

## Основной API

- `useGraphStore()` — получение стора
- `setSearch(search)` — обновить поиск
- `setFilterField(field, value)` — обновить поле фильтра
- `toggleFilterValue(field, value)` — включить/выключить значение в массиве фильтра
- `resetFilters()` — сбросить фильтры
- `selectNode(nodeId)` — выбрать узел
- `clearSelection()` — снять выделение
- `setActiveTab(tabId)` — переключить вкладку карточки
- `toggleSidebar()` — свернуть/развернуть сайдбар

## Запуск

```bash
npm install
npm run dev
```
